Griddyup!

A super tiny CSS grid framework built on CSS grid.

Q: If we already have CSS grid, why do we need a grid framework?

A: Because it gets old repeating the same styles for various grids, and this framework is only 1.88kb UNZIPPED.

Tell the grid how many columns it should be

<div class="display-grid columns-2">
  <div class="bg1">1</div>
  <div class="bg2">2</div>
</div>
